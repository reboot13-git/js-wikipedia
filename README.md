# Wikipedia API

Node js Application

[Download Node.js](https://nodejs.org/en/)

### Run Commands in Command Prompt

`cd <wiki.js file location>`

Install dependencies:

`npm install node-fetch`

`npm install prompt-sync`

Run Wiki.js

`node wiki`
